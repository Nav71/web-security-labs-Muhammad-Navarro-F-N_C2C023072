Run instructions:

- Install Python 3.8+ and create virtualenvs for safety.
- Install requirements from requirements.txt (in project root).

Vulnerable app (port 5001):
  cd vulnerable
  python3 -m venv venv && source venv/bin/activate
  pip install -r ../requirements.txt
  export FLASK_APP=app.py
  flask run --port 5001 --host=127.0.0.1

Secure app (port 5002):
  cd secure
  python3 -m venv venv && source venv/bin/activate
  pip install -r ../requirements.txt
  export FLASK_APP=app.py
  flask run --port 5002 --host=127.0.0.1

Use a browser to visit:
  http://127.0.0.1:5001/   (vulnerable)
  http://127.0.0.1:5002/   (secure)
